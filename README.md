# 🕰️ Time Capsule – Django Backend Project

A secure Django-based web application where users can create **time-locked messages (time capsules)** that can only be accessed after a specified future date. This project focuses on backend development, REST APIs, authentication, and real-world system design.

---

## 🚀 Features

- ⏳ Create time-locked messages (Time Capsules)
- 🔐 User authentication system (secure access control)
- 📬 Email notification when capsule unlocks
- 🔗 Share capsules using unique secure links
- 🧾 REST API endpoints for all major operations
- 🧪 API testing using Postman

---

## 🛠️ Tech Stack

- Backend: Django, Django REST Framework  
- Database: SQLite (development)  
- Authentication: Token Authentication  
- Tools: Postman, Python, Git, GitHub
